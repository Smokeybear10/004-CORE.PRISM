Shortcomings

What we’d do with more time.

The framework is the deliverable. The data and tuning are extensions — here’s what comes next.

  • 01
    More signals, finer time grain.
    Sector-specific newswires, intraday move detection, an expanded universe beyond the focal five tickers.
  • 02
    Train the model to think like an investor.
    A real eval & benchmark loop, then an RL pass that rewards the model for closing the gap with the human-grader baseline.
  • 03
    Close the loop with live trading data.
    Track every lean / fade call as a live-paper position, then score the model against realized PnL the next morning — not against a backtest.

The pipeline is wired. Every box on this deck is a place we can plug a better source in — that’s the point.
